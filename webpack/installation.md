# Installation

## References
 - [official installation guide](https://webpack.js.org/guides/installation/)

## Prerequisites
 - [node.js](https://nodejs.org/)

## Installation
 - webpack & webpack-cli
   ```
   $ cd {your_working_dir}
   $ npm install --save-dev webpack
   $ npm install --save-dev webpack-cli
   ```
 - version check
   ```
   $ npx webpack --version
   4.42.0
   ```
